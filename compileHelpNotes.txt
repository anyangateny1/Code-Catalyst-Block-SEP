(Note: I was using ubuntu Linux in VsCode)

1 - Install the Windows cross-compiler (MinGW-w64):
sudo apt update
sudo apt install mingw-w64

2 - Compile all your .cpp files into a single Windows .exe:
x86_64-w64-mingw32-g++ -std=c++17 -O2 -static -static-libstdc++ -static-libgcc \
  -o block_model.exe main.cpp block.cpp block_growth.cpp block_model.cpp

3 - Zip the executable with no folders, and name the zip after the exe:
zip -j block_model.exe.zip block_model.exe

5 - Submit block_model.exe.zip
